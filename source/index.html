Frequently Asked Questions

How do I add custom CSS to Kirka?

There are two ways to add custom CSS to Kirka:


Option 1: Browser Extension

Install the Stylus extension from Chrome Web Store or Firefox Add-ons. After installation, start the extension, click "Manage," and a new page will open. In the left menu under Style Manager, click the "+" icon. Paste your CSS code into the text field, enter a name for your style in the name field (marked with "?" in the left bar), and click "Save." Close the window, navigate to Kirka, and open the Stylus extension menu. Enable the checkbox for your CSS style and activate the extension (if not already done).


Option 2: Use a Kirka Client

Download any kirka client from KCH Clients list. Once installed follow the guide of the client.

Will customizations affect my fps?

It depends on the specific customization:

  • CSS themes with simple changes generally have minimal to no impact on performance.
  • CSS themes with many animations or effects might slightly reduce FPS on low-end devices.
  • Crosshairs ,Textures, Skyboxes have virtually no impact on game performance.
  • performance impact due to scripts is entirely dependent on the script itself.
Are customizations safe to use?
CSS , Crosshairs ,Textures & Skyboxes are 100% safe to use.

However, be cautious with 3rd party clients and scripts. Always download them from trusted sources and avoid anything that promises unfair advantages.
