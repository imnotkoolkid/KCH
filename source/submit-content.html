Want your CSS themes, crosshairs, textures, clients, scripts, skyboxes, kill icons or sounds hosted on Kirka Community Hub? Contact imnotkoolkid on Discord with your submission, check the requirements for your content type below before reaching out.

Read the ToS
For paid and showcase submissions, you don’t need to include a .css file. For paid themes, users will contact you directly on Discord to purchase. Free submissions must include either the .css file or a GitHub raw link.
Submission Requirements
  • CSS Code — A .css file or direct GitHub raw link. No locally loaded assets.
  • Tags — Three relevant tags (e.g. minimal, dark, cyberpunk).
  • Availability Tag — One of: free, paid, or showcase.
  • Unique Name — A distinct, original name for your theme.
  • Short Description — A concise summary of your CSS theme. (optional)
  • Preview Screenshots — One home screen image and one in-game image.
  • Discord Contact — Format: https://discord.com/users/your_user_id
  • Username — A unique username to identify your submission.
