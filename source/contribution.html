Kirka Community Hub is a solo, open source project by imnotkoolkid a hub for everything Kirka.io . Every contribution, big or small, keeps this place alive and growing.

About the project

What is Kirka Community Hub?

Kirka Community Hub is a community-driven website built to be the go-to resource providing website for the Kirka.io. It shares a catalogue of custom CSS themes, crosshairs, textures, clients ,scripts , skyboxes , kill icons and more shared resources by the community.

The entire codebase is open source and available on GitHub.

How to contribute

Every bit helps

This project grows because people add to it. If you’ve built something cool, found a bug, or just have an idea you can help out.

Submit CSS Made a custom CSS theme for Kirka.io? Submit it to the catalogue, or help keep older themes working with the latest version of the game.
Help build the site Front-end work, bug fixes, new features, refactors are open for pull requests on GitHub.
Report bugs or issues Spotted something broken or weird? Open an issue on the GitHub repo so it can be tracked and fixed or report it on discord.
Suggest ideas Have a feature request or idea for what the hub should include? Drop it as a GitHub issue or reach out on Discord.
Spread the word Share the hub with other Kirka players. Growing the community is one of the most impactful things you can do.

Get started

Ready to help out?

The GitHub repo is the best place to start for code contributions browse open issues, fork the project, and submit a pull request. For content submissions like CSS, crosshairs, textures, clients, scripts, skyboxes, kill icons and more head to the Submit Content page. If you want to get in touch directly, message imnotkoolkid on Discord.
