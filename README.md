## Contributing

Contributions of any size are welcome :) There are a few ways to help out:

**Code contributions** - Bug fixes, new features, refactors, and front-end improvements.

**Content submissions** - If you have a CSS theme, crosshair, texture pack, script, skybox, kill icon, map, or client you want added to the catalogue, use the [Submit Content](https://kirkacommunityhub.pages.dev/submit-content.html) page.

**Bug reports and ideas** - Open an issue on this repo to report bugs or suggest features. you can also reach out to imnotkoolkid directly on Discord.

**Spread the word** - Sharing the hub with other Kirka players helps the catalogue grow.
